Bienvenido a ADAPP-EDU

RESUMEN
ADAPP-EDU es una aplicación que esta en su primera fase de desarrollo, esta fue desarrollada para generar cuestionarios con dificultad adaptativa según rangos, estáticos. El objetivo es tener un cuestionario que según el rendimiento del alumno vaya variando la dificultad permitiendo manejar los niveles de frustración durante el uso de la aplicación.

INSTALACIÓN
1)Descargue la carpeta del proyecto con todos sus archivos.
2)Copie los archivos en el servidor que desee correr la aplicación.
3)En el archivo ADAPP-EDU.js reemplace la variable de URL que en este momento es "https://adappedu.lircaytech.com" por la dirección URL donde este corriendo el servicio de base de datos.

Una vez que tenga corriendo el servicio de preguntas y el servicio para ver la preguntas, cree un usuario usando la pagina principal del sistema.
